# **App Name**: TradeVision

## Core Features:

- Trade Entry Interface: Intuitive interface for real-time or retrospective trade logging, accessible on both mobile and desktop devices.
- Strategy Flexibility: Allow creation, modification, and deletion of trading strategies to categorize and analyze trades.
- Summary Dashboard: Dashboard providing total P&L, win/loss ratio, risk:reward stats, and most/least profitable strategies, plus an AI powered tool suggesting ways to interpret your summary statistics in the light of current events.
- Export to CSV: Enable report downloads in CSV format.

## Style Guidelines:

- Primary color: HSL(210, 65%, 50%) - A deep sky blue (#3391FF) evoking trust and stability.
- Background color: HSL(210, 20%, 95%) - A very light desaturated sky blue (#F0F5FA) providing a clean and professional backdrop.
- Accent color: HSL(180, 55%, 40%) - A vibrant turquoise (#3BAF9F) for interactive elements, buttons, and key performance indicators.
- Body: 'PT Sans' (sans-serif) for the body; Headline: 'Playfair' (serif) for headers. 'Source Code Pro' (monospace) for displaying code snippets, like strategy names.
- Use consistent and professional icons related to trading, charts, and data analysis.
- Use a clean and structured layout with clear sections for data entry, reporting, and analysis.
- Use subtle transitions and animations for loading data and updating charts to enhance user experience.
